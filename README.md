# Hierarchical Argparse

## Usage

+ See ```/examples/example.py```
+ You can use hiargparse with normal (original) argparse. See ```/examples/example_with_original.py```.

## Requirements

+ python >= 3.6.1
+ typing_extensions (for typing\_extentions.Protocol in python 3.6.5)
